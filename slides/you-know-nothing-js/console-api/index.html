Basics

What is the Console Api?

It's an Api to allow display information (for debugging purpose) into the console

Basically, this is a logger

We have basics logger methods

  • console.debug (only be displayed if the console is configured to verbose mode)
  • console.log
  • console.info
  • console.warn
  • console.error (or console.exception)

In motion

console.clear

Clear the console

console.assert

We define an assertion and an error will appear with the stackstrace if the assertion is not respected

Less basics

console.trace

Display as a tree the specified JavaScript object

console.dirxml

Display as a tree the specified XML or HTML element

console.table

Display as a table to illustrate

  • a simple array
  • a complex array
  • an object

Count

console.count

Display the number of time the method is called

Can have a label as parameter (default value: "default")

Example

            function count(target, key, descriptor) {
  const myMethod = descriptor.value;
  descriptor.value = function (...arg) {
    console.count(myMethod.name);
    return myMethod.apply(this, arg);
  };
  return descriptor;
}

class MyClass {
  @count
  myAwesomeFunction() {
    const randomColor = Math.floor(Math.random()*16777215).toString(16);
    console.info('%cNo really awesome...', `color:#${randomColor};`);
  }
};

In motion

                const instance = new MyClass();
const howManyTime = Math.ceil(Math.random() * 10);
console.info(
  'We should call the method %s times',
  howManyTime
);

for (let i = 0; i < howManyTime; ++i) {
  instance.myAwesomeFunction();
}

console.countReset

Reset the number of time the method is called

Can have a label as parameter (default value: "default")

Timing

console.time / console.timeEnd

Will display the number of time between the call of the two methods

Can have a label as parameter (default value: "default")

Example (as annotation)

            function timing(target, key, descriptor) {
  const myMethod = descriptor.value;

  descriptor.value = function (...arg) {
  console.time(myMethod.name);
    const returned = myMethod.apply(this, arg);

    if (returned instanceof Promise) {
    	return returned.then((result) => {
        console.timeEnd(myMethod.name);
      	return result;
      });
    }

    console.timeEnd(myMethod.name);
    return returned;
  };
  return descriptor;
}

Example (usage)

            class MyClass {
  @timing
  myAwesomeFunction() {
    const randomColor = Math.floor(Math.random()*16777215).toString(16);
    console.info('%cNo really awesome...', `color:#${randomColor};`);
  }

  @timing
  myAsyncFunction() {
  	return new Promise((resolve) => {
    	setTimeout(() => {
        this.myAwesomeFunction();
        resolve();
      }, 150);
    });
  }
}

In motion

                (async function() {
  const instance = new MyClass();
  instance.myAwesomeFunction();
  await instance.myAsyncFunction();
})();
